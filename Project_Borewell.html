Real-Time Borewell Monitoring System

Developed a deep borewell monitoring system to measure underground water levels in real-time for drought-prone regions. The system used a string-tension based sensing mechanism with precise voltage calibration to measure depth variations.

GSM communication enabled remote data transmission to a central server. Implemented automated connectivity checks, validation callbacks, and a user-facing display interface to help users monitor seasonal water table changes.
